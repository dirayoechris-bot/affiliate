# Yoga Path Guide — Affiliate Blog

Affiliate marketing blog for yoga teacher training programs.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npx vercel
```

## Add Blog Posts

Add markdown files to `content/blog/` with frontmatter:

```md
---
title: "Your Title"
date: "2026-05-20"
excerpt: "Short description"
---

Content here...
```
